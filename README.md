E-Commerce Backend System (Python CLI)
📌 Project Overview

This is a Python-based CLI E-Commerce Backend System that simulates real-world online shopping functionality. It allows users to browse products, manage a shopping cart, and place orders with automatic order tracking and history storage.

The project focuses on backend logic using Python data structures and file handling (JSON).

🚀 Features
🛍️ Product Management
View available products
Each product includes ID, name, price, category, and stock
🛒 Shopping Cart
Add products to cart
Remove items from cart
Update quantities
Automatic total calculation
💳 Order Processing
Place orders from cart
Generate unique order ID
Deduct stock after purchase
Clear cart after order
📦 Order History
Stores all orders in orders.json
Displays:
Order ID
Items purchased
Total amount
Date & time
⚙️ System Features
Input validation
Stock verification
Error handling
Modular function-based structure
🧑‍💻 Technologies Used
Python 3
JSON (for data storage)
UUID (for order ID generation)
Datetime module
